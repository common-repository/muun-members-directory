== Description ==
Get an always up to date member directory for your community.

Showcase the members of your community on your WordPress site. Management of your members is easily done through your Muun dashboard. Members can be ordered in various ways (defaults to last added) and you can change the style of your overview to match your branding.

Installation is easy and all you need is your space id ready. A monthly subscription for Muun is needed.

== Installation ==
1. In your WordPress admin panel, go to Plugins > New Plugin, search for Muun members directory and click 'Install now'
2. Alternatively, download the plugin and upload the contents of muun-members-directory.zip to your plugins directory, which usually is /wp-content/plugins/.
3. Activate the plugin
4. Go to Settings > Muun members directory
5. Set your Space ID key in the plugin settings.
6. Embed on any page or posts by using the [muun-members-directory] shortcode.
